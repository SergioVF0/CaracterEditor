# ✍️ CaracterEditor

Un editor de texto ligero desarrollado en **Java** con interfaz gráfica (**Swing**) diseñado para realizar operaciones rápidas de edición de caracteres.

## ✨ Funcionalidades
- **Sustitución:** Reemplaza caracteres específicos por otros de forma masiva.
- **Eliminación:** Borra caracteres seleccionados del texto cargado.
- **Interfaz Gráfica:** Uso de la librería Swing para una experiencia de usuario sencilla.

## 🛠️ Tecnologías utilizadas
- **Lenguaje:** Java
- **Librería de UI:** Swing (Java Foundation Classes)

## 📦 Contenido del Repositorio
- Código fuente en archivos `.java`.
- Ejecutable comprimido en `.rar`.
- Archivo ejecutable `.jar` y `.exe` para uso directo.

## 🚀 Cómo ejecutar
Si tienes instalado el JRE (Java Runtime Environment), puedes ejecutar el proyecto directamente usando el archivo `.jar`:

1. Descarga el archivo `CaracterEditor.jar`.
2. Ejecútalo desde la terminal o haciendo doble clic:
   ```bash
   java -jar CaracterEditor.jar
